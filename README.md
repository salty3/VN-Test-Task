Card Game implementation taken from my [another test task.](https://github.com/salty3/Card-Flip-Test-Task)

1. Open the `Init` scene.
2. Press play.

### [SmoothLayout](https://gist.github.com/codorizzi/79aab1ae7d7940fe3e3603af61cd8617)
